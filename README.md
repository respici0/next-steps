# Next Steps 

A job tracking app with features that I think would be useful.


This project was created in collaboration with

[![GitHub](https://img.shields.io/badge/GitHub-treysouz-181717?style=flat&logo=github)](https://github.com/treysouz)


## Running locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies

- Next.js
- Vercel
- MongoDB
- Mongoose
- Shadcn/ui
